Code and Build

Website Artifacts

  • index.html: async invoke Web API to display user & server IP addresses.
  • favicon.ico: the website icon.

Lambda Function for Web API

Java Classes

  • MyIpApp: the Lambda handler class to retrieve user and server IP addresses.
  • MyIpRequest: to wrap the request.
  • MyIpResponse: to wrap the response.
  • ServerIpNotFoundError: to wrap the inner exception when something wrong when retrieving server IP.
  • UserIpNotFoundError: to wrap the error message when something wrong when retrieving user IP.

Unit Tests

  • MyIpAppTest: the unit test class.
  • Coverage:
    • Class: 80%
    • Method: 88.9%
    • Line: 84.6%
    • Coverage report

How to Build It?

Use the following Maven command to run all unit tests and release a build.

mvn package
